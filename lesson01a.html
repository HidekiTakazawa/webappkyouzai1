DOM操作の基本

プログラムの仕様

HTML

pタグを作る。内容は何でもよいが後で中国語の音声出力をするように変更したいので中国語文章を入力する。javascriptで操作できるようにするためにidを設定する。仮にp1と命名する。

CSS

idがp1に対して文字サイズを24px cssで設定。

中国語の文にカーソルを置くとポインターが手の形になる。

Javascript

定数p1を用意し定数p1にgetElementByIdメソッドでHTMLのpタグをセットする。

定数p1にclickイベントを設定する。 addEventListenerメソッドを使用する。

中国語の文をclickしたら文字色を青色にする。

青色にする方法は2種類ある。 定数p1のstyle.colorプロパティにblueをセット　又は CSSに文字色が青色のクラスを作り、定数p1.classList.addで青色のクラスをセットする。
