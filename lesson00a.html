Live Server は、Visual Studio Code（VSCode）の拡張機能です。テキストエディタの変更内容をリアルタイムでブラウザに反映する機能で、簡易的なローカルサーバー環境を実現します。 Live Server は、フロントエンド開発には欠かせない人気のツールです。ファイルを手動でリロードしなおすことなく、反映を確認できるので、効率的な開発ができます。
