DOM操作の基本 リスト要素のulタグにliタグを動的に追加する。

htmlの記述

<label for="todo">TODO input:</label>
<input type="text" id="todo" name="todo">
<input type="button" id="btn" value="追加">
<ul id="ul1"> </ul>

入力欄のinputboxのidをtodoにした。 追加のボタンのidをbtn にした。 ulタグのidをul1とした。ulタグの子供のliタグの記述はない。これはjavascript で動的に追加する。

CSSの記述

#btn { display: inline-block; width: 100px; height: 40px; } 追加ボタンはinputタグで定義した。inputタグはinline要素なのでwidth指定ができない。だからdisplayをinline-blockにした。

Javascriptの記述。

const btn = document.getElementById('btn');
btn.addEventListener('click', () => {
const ul1 = document.getElementById('ul1');
const li = document.createElement('li');
const todo = document.getElementById('todo');
li.textContent = todo.value;
ul1.appendChild(li);
});

btnのDOMを取得して定数のbtnを作成する。getElementByIdを使用。 btnのクリックイベントを定義する。 ul1のDOMを取得する。li要素を新規作成し定数のliを作成する。 todoのDOMを取得して定数のtodoを作成する。 liのtextContentプロパティにtodoのvalueプロパティの値をセットする。 ur1のappendChildメソッドを使いliをur1に追加する。
